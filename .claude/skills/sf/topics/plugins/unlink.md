# plugins:unlink

## Description

Removes a plugin from the CLI.

## Usage

```bash
sf plugins:unlink [flags]
```

## Flags

### Command Flags

### `-h, --help`

Show CLI help.

- **Type**: boolean

### `-v, --verbose`

- **Type**: boolean

## Examples

sf plugins:unlink <%- config.pjson.oclif.examplePlugin || "myplugin" %>

## Aliases

- `plugins:unlink`
- `plugins:remove`

## Alternative Syntaxes

- `sf plugins:uninstall`
- `sf uninstall:plugins`

## Plugin

**@oclif/plugin-plugins** (core)
