# plugins:discover

**See a list of 3rd-party sf plugins you can install.**

## Usage

```bash
sf plugins:discover [flags]
```

## Flags

### Global Flags

### `--flags-dir`

Import flag values from a directory.

- **Type**: option

### `--json`

Format output as json.

- **Type**: boolean

## Examples

sf plugins:discover

## Alternative Syntaxes

- `sf discover:plugins`
