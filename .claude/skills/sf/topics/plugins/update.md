# plugins:update

## Description

Update installed plugins.

## Usage

```bash
sf plugins:update [flags]
```

## Flags

### Command Flags

### `-h, --help`

Show CLI help.

- **Type**: boolean

### `-v, --verbose`

- **Type**: boolean

## Alternative Syntaxes

- `sf update:plugins`

## Plugin

**@oclif/plugin-plugins** (core)
