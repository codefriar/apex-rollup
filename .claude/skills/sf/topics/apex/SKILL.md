# sf apex

**Topic for apex-related operations**

## Overview

This topic contains 8 command(s) related to apex operations in Salesforce.

## Commands

### [apex:generate:class](./generate-class.md)

Generate an Apex class.

### [apex:generate:trigger](./generate-trigger.md)

Generate an Apex trigger.

### [apex:get:log](./get-log.md)

Fetch the specified log or given number of most recent logs from the org.

### [apex:get:test](./get-test.md)

Display test results for a specific asynchronous test run.

### [apex:list:log](./list-log.md)

Display a list of IDs and general information about debug logs.

### [apex:run](./run.md)

Execute anonymous Apex code entered on the command line or from a local file.

### [apex:run:test](./run-test.md)

Invoke Apex tests in an org.

### [apex:tail:log](./tail-log.md)

Activate debug logging and display logs in the terminal.

## Quick Reference

```bash
sf apex:generate:class
sf apex:generate:trigger
sf apex:get:log
sf apex:get:test
sf apex:list:log
sf apex:run
sf apex:run:test
sf apex:tail:log
```
