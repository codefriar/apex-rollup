# sf lightning

**Topic for lightning-related operations**

## Overview

This topic contains 5 command(s) related to lightning operations in Salesforce.

## Commands

### [lightning:generate:app](./generate-app.md)

Generate a Lightning App.

### [lightning:generate:component](./generate-component.md)

Generate a bundle for an Aura component or a Lightning web component.

### [lightning:generate:event](./generate-event.md)

Generate a Lightning Event.

### [lightning:generate:interface](./generate-interface.md)

Generate a Lightning Interface.

### [lightning:generate:test](./generate-test.md)

Generate a Lightning test.

## Quick Reference

```bash
sf lightning:generate:app
sf lightning:generate:component
sf lightning:generate:event
sf lightning:generate:interface
sf lightning:generate:test
```
