# sf force

## Overview

This topic contains 7 command(s) related to force operations in Salesforce.

## Commands

### [force:data:bulk:delete](./data-bulk-delete.md)

Bulk delete records from an org using a CSV file. Uses Bulk API 1.0.

### [force:data:bulk:status](./data-bulk-status.md)

View the status of a bulk data load job or batch. Uses Bulk API 1.0.

### [force:data:bulk:upsert](./data-bulk-upsert.md)
