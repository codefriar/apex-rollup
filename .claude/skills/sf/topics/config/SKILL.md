# sf config

**Topic for config-related operations**

## Overview

This topic contains 4 command(s) related to config operations in Salesforce.

## Commands

### [config:get](./get.md)

Get the value of a configuration variable.

### [config:list](./list.md)

List the configuration variables that you've previously set.

### [config:set](./set.md)

Set one or more configuration variables, such as your default org.

### [config:unset](./unset.md)

Unset local or global configuration variables.

## Quick Reference

```bash
sf config:get
sf config:list
sf config:set
sf config:unset
```
