# sf alias

**Topic for alias-related operations**

## Overview

This topic contains 3 command(s) related to alias operations in Salesforce.

## Commands

### [alias:list](./list.md)

List all aliases currently set on your local computer.

### [alias:set](./set.md)

Set one or more aliases on your local computer.

### [alias:unset](./unset.md)

Unset one or more aliases that are currently set on your local computer.

## Quick Reference

```bash
sf alias:list
sf alias:set
sf alias:unset
```
