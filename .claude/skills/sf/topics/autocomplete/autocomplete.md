# autocomplete

## Description

Display autocomplete installation instructions.

## Usage

```bash
sf autocomplete [flags]
```

## Flags

### Command Flags

### `-r, --refresh-cache`

Refresh cache (ignores displaying instructions)

- **Type**: boolean

## Examples

$ sf autocomplete

$ sf autocomplete bash

$ sf autocomplete zsh

$ sf autocomplete powershell

$ sf autocomplete --refresh-cache

## Plugin

**@oclif/plugin-autocomplete** (core)
