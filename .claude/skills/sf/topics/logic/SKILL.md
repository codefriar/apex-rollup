# sf logic

**Topic for logic-related operations**

## Overview

This topic contains 2 command(s) related to logic operations in Salesforce.

## Commands

### [logic:get:test](./get-test.md)

Get the results of a test run.

### [logic:run:test](./run-test.md)

Invoke tests for Apex and Flows in an org.

## Quick Reference

```bash
sf logic:get:test
sf logic:run:test
```
