# sf community

**Topic for community-related operations**

## Overview

This topic contains 3 command(s) related to community operations in Salesforce.

## Commands

### [community:create](./create.md)

Create an Experience Cloud site using a template.

### [community:list:template](./list-template.md)

Retrieve the list of templates available in your org.

### [community:publish](./publish.md)

Publish an Experience Builder site to make it live.

## Quick Reference

```bash
sf community:create
sf community:list:template
sf community:publish
```
