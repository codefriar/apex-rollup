# sf sobject

**Topic for sobject-related operations**

## Overview

This topic contains 2 command(s) related to sobject operations in Salesforce.

## Commands

### [sobject:describe](./describe.md)

Display the metadata for a standard or custom object or a Tooling API object.

### [sobject:list](./list.md)

List all Salesforce objects of a specified category.

## Quick Reference

```bash
sf sobject:describe
sf sobject:list
```
