# commands

## Description

List all sf commands.

## Usage

```bash
sf commands [flags]
```

## Flags

### Command Flags

### `-c, --columns`

Only show provided columns (comma-separated).

- **Type**: option
- **Multiple**: Can be specified multiple times

### `--deprecated`

Show deprecated commands.

- **Type**: boolean

### `-x, --extended`

Show extra columns.

- **Type**: boolean

### `--hidden`

Show hidden commands.

- **Type**: boolean

### `--no-truncate`

Do not truncate output.

- **Type**: boolean

### `--sort`

Property to sort by.

- **Type**: option
- **Default**: `id`

### `--tree`

Show tree of commands.

- **Type**: boolean

### Global Flags

### `--json`

Format output as json.

- **Type**: boolean

## Plugin
