# sf code-analyzer

**Topic for code-analyzer-related operations**

## Overview

This topic contains 3 command(s) related to code-analyzer operations in Salesforce.

## Commands

### [code-analyzer:config](./config.md)

Output the current state of configuration for Code Analyzer.

### [code-analyzer:rules](./rules.md)

List the rules that are available to analyze your code.

### [code-analyzer:run](./run.md)

Analyze your code with a selection of rules to ensure good coding practices.

## Quick Reference

```bash
sf code-analyzer:config
sf code-analyzer:rules
sf code-analyzer:run
```
