# sf cf

**Topic for cf-related operations**

## Overview

This topic contains 6 command(s) related to cf operations in Salesforce.

## Commands

### [cf:build:scratch:step1](./build-scratch-step1.md)

Create or identify a scratch org for deployment.

### [cf:deploy:pr](./deploy-pr.md)

Deploy a GitHub Pull Request to a Salesforce org

### [cf:erd](./erd.md)

Generate an Entity Relationship Diagram (ERD) from local Salesforce metadata using Mermaid syntax.

### [cf:logs:delete](./logs-delete.md)

Delete Apex debug logs from a Salesforce org.

### [cf:logs:download](./logs-download.md)

Download Apex debug logs from a Salesforce org.

### [cf:logs:list](./logs-list.md)

List Apex debug logs from a Salesforce org.

## Quick Reference

```bash
sf cf:build:scratch:step1
sf cf:deploy:pr
sf cf:erd
sf cf:logs:delete
sf cf:logs:download
sf cf:logs:list
```
