# sf schema

**Topic for schema-related operations**

## Overview

This topic contains 4 command(s) related to schema operations in Salesforce.

## Commands

### [schema:generate:field](./generate-field.md)

Generate metadata source files for a new custom field on a specified object.

### [schema:generate:platformevent](./generate-platformevent.md)

Generate metadata source files for a new platform event.

### [schema:generate:sobject](./generate-sobject.md)

Generate metadata source files for a new custom object.

### [schema:generate:tab](./generate-tab.md)

Generate the metadata source files for a new custom tab on a custom object.

## Quick Reference

```bash
sf schema:generate:field
sf schema:generate:platformevent
sf schema:generate:sobject
sf schema:generate:tab
```
